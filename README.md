# Portfolio (Next.js + TypeScript)

This project is migrated from Angular to Next.js App Router with a TypeScript-only codebase and preserved portfolio content.

## Scripts

- `npm start` or `npm run dev`: start local dev server
- `npm run build`: production build
- `npm run lint`: run ESLint
- `npm test`: placeholder command (tests intentionally skipped in first migration pass)

## Notes

- Static assets are served from `public/`.
- Main page composition lives in `src/app/page.tsx`.
- Shared data and typings live in `src/lib/`.
